# Nyz

## What is Nyz?
Nyz is a blogging platform where you can share content about anything that you think its interesting to share with other people in the community.

## Dependencies
<ul>
  <li>ReactJs</li>
  <li>NodeJs v18.12.1</li>
  <li>Java 11</li>
  <li>Spring Framework</li>
  <li>PostgreSQL 15</li>
</ul>
